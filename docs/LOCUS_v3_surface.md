# LŌCUS v3 — "어제의 나를 이기기"

> 못 했으면, 지금 할 수 있는 가장 작은 버전을 제안해준다.

## 1. 겉 (사용자가 보는 것)

### 핵심 경험
- 개인 목표를 던져둔다 ("운동하기", "영어 공부", "팀장한테 의견 말하기")
- 하루 중 LŌCUS가 물어본다: "했어?"
- 못 했으면: "지금 이것만 할까요?" — 가장 작은 버전 제안
- 했으면: 조용히 기록됨
- 며칠 쌓이면: 패턴이 보인다 ("화요일에 자주 못 함", "외부 일정 있는 날 포기함")

### 사용자 흐름

```
[목표 설정]
"운동하기" 던짐 (한 줄)
↓
[하루 중 — 공 던지기]
막히면 탭 한 번 (텍스트 없이)
→ 나중에 "뭐였어?" 물어봄
↓
[하루 끝 — 체크인]
LŌCUS: "운동 했어?"
→ 예: "오늘도 이겼네요." 끝.
→ 아니오: "지금 스트레칭 1분만 할까요?"
    → 했어: "작게라도 한 거예요."
    → 안 할래: 조용히 기록됨. 판단 없음.
↓
[주간]
"이번 주 5일 중 3일 이겼어요."
"화요일에 자주 못 지키고 있어요."
"항상 외부 일정(통제불가) 있는 날 포기해요."
```

### 원칙
- "못 했네요"는 절대 안 함. 판단 없음.
- 작은 대안 제안은 조언이 아니라 실용적 옵션
- 안 해도 됨. LŌCUS는 확인하지 않음.
- 하루 못 열어도 죄책감 없음.

## 2. 속 (엔진이 하는 것)

| 겉에서 보이는 것 | 엔진이 하는 것 |
|---|---|
| 목표 분류 (건강/일/관계) | Domain 자동 분류 |
| "화요일에 자주 못 함" | 시간 패턴 + recurring 감지 |
| "외부 일정 때문" | Direction: out (통제불가) 감지 |
| "이번 주 3일 이김" | 주간 변화 리포트 |
| "이 목표 2주째 미결" | Nature: unresolved 추적 |
| "스트레칭 1분만?" | 목표의 작은 버전 제안 (AI) |
| 공 던지기 | 시간/빈도로 스트레스 패턴 감지 |

## 3. "작은 버전 제안" 로직

목표의 작은 버전은 AI가 생성:
- "운동하기" → "스트레칭 1분", "계단 한 층", "산책 5분"
- "영어 공부" → "단어 3개만", "유튜브 영상 하나만"
- "보고서 끝내기" → "목차만 써두기", "첫 문장만 적기"
- "팀장한테 의견 말하기" → "오늘은 질문 하나만 해보기"

원칙:
- 1~3분 안에 끝나는 것
- 실패가 불가능한 수준
- 판단/조언 톤 금지 ("해야 해요" X, "할까요?" O)

## 4. 공 던지기 (텍스트 없는 기록)

- 하루 중 막히거나 무거울 때: 탭 한 번
- 텍스트 없음. 시간만 기록됨.
- 나중에 여유 있을 때 LŌCUS가 물어봄: "오후 3시쯤 뭔가 있었던 것 같은데, 뭐였어?"
- 답하면: 엔진 작동. 안 답해도: 무시됨.

## 5. 기존 엔진과의 연결

- 목표 텍스트 → Weight 4축 자동 분류
- 달성/미달 기록 → 미결 추적
- 반복 실패 패턴 → recurring 감지
- 무게 순위 → 가장 오래 미결인 목표
- 주간 리포트 → "어제의 나를 이긴 날 vs 진 날"
- Core Perspective → "당신은 관계 목표에서 자주 막히고, 건강 목표는 잘 지킵니다."

## 6. 한 문장

> 어제의 나를 이기는 가장 작은 방법을 매일 알려준다.
